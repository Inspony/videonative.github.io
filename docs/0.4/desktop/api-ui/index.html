VN 控件属性和事件

总览

控件属性类型说明：

  • Property：控件的属性。可以直接写在标签中，也可以写在 vnss 规则集中
  • Event：控件的事件。通过将JS方法绑定到组件事件上，可以处理事件回调
  • Method：控件DOM方法，供JS代码调用
